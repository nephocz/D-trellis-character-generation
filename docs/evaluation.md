# 评估记录

## Exp001：Baseline 单图推理初步评估


### 1. 初步结论

-本次 baseline 实验已经完成基础推理输出，可以导入blender，unity等软件，说明当前 AutoDL 环境、TRELLIS.2 调用代码、GLB 导出流程和预览视频渲染流程可以正常运行。

-本次实验用了11张例图，3张AI生成平面插画，3张手绘平面插画，5张不同风格的卡通渲染图片，主要图片都是正面视角图。综合看下来模型都可以大致生成，但是离商业化应用还有很大的距离，主要问题如下：
    1.服装过于复杂时模型无法准确还原
    2.如果原图有透视效果可能导致比例不对
    3.只能输入单张图片，背面信息缺失
    4.模型精度不够，可以看见很多明显的锯齿
    5.模型人物脸部扭曲，效果很差
    6.人物皮肤颜色调的很差

-当前已经可以导出低精度的图片，当游戏中远处的低模没有问题，但是经不起推敲
